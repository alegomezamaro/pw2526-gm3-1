Socio
    -VARIABLES
        String nombre
        String apellidos
        String dni
        LocalDate fechaNacimiento
        String direccion
        Bool patron
    -VISTAS
        Añadir socio (DNI)
        Añadir título de patrón a un socio
        Listar socios

Inscripción
    -VARIABLES
        Int idInscripción
        Int idFamilia
        String dniTitular
        LocalDate fechaInscripción
        Int cuotaAnual (250€ adulto, 100€ niño)
        Enum tipoCuota
            InscripcionIndividual
            InscripcionFamilia (Máximo 2 adultos)
    -VISTAS
        Ampliación inscripción (Se puede pasar de individual a familia)
        Listar inscripciones (Titular con familia si tienen)


Familia
    -VARIABLES
        Int idFamilia
        String dniTitular
        Int numeroAdultos (1-2)
        Int numeroNiños

Embarcacion
    -VARIABLES
        String matricula
        Enum tipoEmbarcacion
            Velero
            Yate
            Catamarán
            Lancha
        String nombre
        Int plazasDisponibles
        Int dimension
        String patron (Se enlaza despues)
    -VISTAS
        Dar alta embarcación (MATRÍCULA (NOMBRE no primaria))
        Asignar un patrón (Si ya tenia uno preguntar si se quiere cambiar)
        Consultar por tipo de embarcación


Patron
    -VARIABLES
        String nombre
        String apellidos
        String dni
        Date fechaNacimiento
        Date fechaExpedicionTitulo
    -VISTAS
        Dar de alta patrón (DNI)

Alquiler
    -VARIABLES
        Int idAlquiler x
        String socioTitular x
        String matricula x
        Int plazasNecesarias x
        Int precio (20€pp) x
        Date fechaInicio x
        Date fechaFin x
    -VISTAS
        Buscar embarcacion disponible (FECHA INICIO-FIN)
        Alquilar embarcación (EMBARCACION DISPONIBLE)
            3 días: Octubre-Abril
            1 o 2 semanas: Mayo-Septiembre
        Listar alquileres

Eventos (Reserva)
    -VARIABLES
        Int idEvento
        String matriculaEmbarcación
        Date fechaEvento (1 dia)
        Int plazasNecesarias
        Int precio (40€pp)
    -VISTAS
        Reservar embarcación (FECHA (PATRÓN asociado))
